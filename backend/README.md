# DevBoard Backend

FastAPI backend for the DevBoard developer command centre application.

## Development

Install dependencies:
```bash
make install
```

Run the application:
```bash
make start
```

## Code Quality

Format, lint, and type check:
```bash
make format
make lint
make type-check
```

Run all checks:
```bash
make all-checks
```

## Database

Run migrations:
```bash
make migrate
```

Generate new migration:
```bash
make migrate-auto
```

## Testing

Run tests:
```bash
make test
```

## Available Commands

Run `make help` to see all available commands.
